Clase:

La Clase es como la receta inicial
Es la fábrica de objetos, tiene las características y métodos generales para todos los objetos que fabriquemos.

Objeto:

El objeto lo creamos a partir de una clase con características generales.
Por ejemplo, el objeto "perro" pertenece a la clase "Animal" por lo tanto

hereda todos sus métodos y atributos

Atributos:

Las características que definen al objeto, por ejemplo, especie, peso, altura, edad, color, etc.

Método

Lo que el objeto es capaz de hacer, por ejemplo, caminar, saltar, ladrar, comer, jugar, etc.

Constructor:

Una función OBLIGATORIA, cuando creamos una clase, debemos crear un constructor

que es el que referencia como se debe crear el objeto
